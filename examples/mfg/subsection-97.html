Subsection 5.4 Radical Notation

Because \(a^{1/n} = \sqrt[n]{a}\text{,}\) we can write any power with a fractional exponent in radical form as follows.

Rational Exponents and Radicals.
\begin{equation*} a^{m/n} = \sqrt[n]{a^m} =\left( \sqrt[n]{a}\right)^m \end{equation*}
Example 5.4.8.

  1. \(125^{4/3} = \sqrt[3]{125^4} \text{ or } \left(\sqrt[3]{125}\right)^4\)

  2. \(x^{0.4} = x^{2/5} = \sqrt[5]{x^2}\)

  3. \(6w^{-3/4} = \dfrac{6}{\sqrt[4]{w^3}}\)

Checkpoint 5.4.9.

Write each expression in radical notation.

  1. \(5t^{1.25}\)

  2. \(3m^{-5/3}\)

Answer

  1. \(5\sqrt[4]{t^5} \)

  2. \(\dfrac{3}{\sqrt[3]{m^5} } \)

Usually, we will want to convert from radical notation to fractional exponents, since exponential notation is easier to use.

Example 5.4.10.

  1. \(\sqrt{x^5} = x^{5/2}\)

  2. \(5 \sqrt[4]{p^3} = 5p^{3/4}\)

  3. \(\dfrac{3}{\sqrt[5]{t^2}}= 3t^{-2/5}\)

  4. \(\sqrt[3]{2y^2} = \left(2y^2\right)^{1/3} = 2^{1/3} y^{2/3}\)

Checkpoint 5.4.11.

Convert to exponential notation.

  1. \(\sqrt[3]{6w^2}\)

  2. \(\sqrt[4]{\dfrac{v^3}{s^5}}\)

Answer

  1. \(6^{1/3} w^{2/3} \)

  2. \(v^{3/4} s^{-5/4} \)
