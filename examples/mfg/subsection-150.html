Subsection 3.4 Logarithmic Equations

A logarithmic equation is one in which the variable appears inside of a logarithm. For example,

\begin{equation*} \log_4 x = 3 \end{equation*}

is a log equation. To solve a log equation, remember that logarithms and exponentials with the same base are inverse functions. Therefore,

\begin{equation*} \blert{y = \log_{b}{x}} ~~~~\text{ if and only if }~~~~ \blert{x = b^y} \end{equation*}

Thus, we can rewrite a logarithmic equation in exponential form.

Example 3.4.11.

Solve for \(x\text{.}\)

  1. \(\displaystyle 2(\log_3 x) - 1 = 4\)
  2. \(\displaystyle \log_{10}{(2x + 100)} = 3\)
Solution

  1. We isolate the logarithm, then rewrite the equation in exponential form:

    \begin{equation*} \begin{aligned}[t] 2(\log_3 x) \amp = 5\amp\amp \blert{\text{Divide both sides by 5.}}\\ \log_3 x \amp = \frac{5}{2}\amp\amp \blert{\text{Convert to exponential form.}}\\ x \amp = 3^{5/2} \end{aligned} \end{equation*}

  2. First, we convert the equation to exponential form.

    \begin{equation*} 2x + 100 = 10^3 = 1000 \end{equation*}

    Now we can solve for \(x\) to find \(2x = 900\text{,}\) or \(x = 450\text{.}\)

Checkpoint 3.4.12.

Solve for the unknown value in each equation.

  1. \(\displaystyle \log_b 2 = \dfrac{1}{2}\)
  2. \(\displaystyle \log_{3}{(2x - 1)} = 4\)
Answer

  1. \(b=4\)

  2. \(x=41\)

Example 3.4.13.

If \(f(x) = \log_{10}{x}\text{,}\) find \(x\) so that \(f(x)=-3.2\text{.}\)

Solution

We must solve the equation \(\log_{10}{x} = -3.2\text{.}\) Rewriting the equation in exponential form yields

\begin{equation*} x = 10^{-3.2}\approx 0.00063 \end{equation*}
Note 3.4.14. Evaluating \(10^x\).

In ExampleÂ 3.4.13, the expression \(10^{-3.2}\) can be evaluated in two different ways with a calculator. We can use the ^ key and press

\(10\) ^ (-) \(3.2\) ENTER

which gives 6.30957344 E -4, or approximately \(0.00063\text{.}\) Alternatively, because \(10^x\) is the inverse function for \(\log x\text{,}\) we can press

2nd LOG (-) \(3.2\) ENTER

which gives the same answer as before.

Checkpoint 3.4.15.

Imagine the graph of \(f(x) = \log_{10}{x}\text{.}\) How far must you travel along the \(x\)-axis until the \(y\)-coordinate reaches a height of \(5.25\text{?}\)

Answer

\(x = 177,827.941\)
