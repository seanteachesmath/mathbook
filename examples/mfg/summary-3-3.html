Subsubsection 4.9.2 CONCEPTS

  1. \(n\)th roots: \(s\) is called an \(n\)th root of \(b\) if \(s^n = b\text{.}\)

  2. Exponential notation: For any integer \(n\ge 2\) and for \(a\ge 0\text{,}\) \(a^{1/n}=\sqrt[n]{a}\text{.}\)

  3. We cannot write down an exact decimal equivalent for an irrational number, but we can approximate an irrational number to as many decimal places as we like.

  4. We can solve the equation \(x^n = b\) by raising both sides to the \(\dfrac{1}{n} \) power.

  5. An allometric equation is a power function of the form \(\text{variable} = k (\text{mass})^p\text{.}\)

  6. We can solve the equation \(x^{1/n} = b\) by raising both sides to the \(n\)th power.

  7. Roots of Real Numbers.

    • Every positive number has two real-valued roots, one positive and one negative, if the index is even.

    • A negative number has no real-valued root if the index is even.

    • Every real number, positive, negative, or zero, has exactly one real-valued root if the index is odd.

Subsubsection 4.9.3 STUDY QUESTIONS

  1. Use an example to illustrate the terms radical, radicand, index, and principal root.

  2. Explain why \(x^{1/4}\) is a reasonable notation for \(\sqrt[4]{x} \text{.}\)

  3. What does the notation \(x^{0.2}\) mean?

  4. Express each of the following algebraic notations in words; then evaluate each for \(= 16\text{:}\)

    \begin{equation*} 4x, ~~x^4, ~~\frac{x}{4}, ~~\frac{1}{4}x, ~~x^{1/4}, ~~x-4, ~~x^{-1/4} \end{equation*}

  5. How is the third law of exponents, \((xa)^b = x^{ab}\text{,}\) useful in solving equations?
