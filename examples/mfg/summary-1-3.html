Subsubsection 4.5.2 CONCEPTS

  1. The point \((a, b)\) lies on the graph of the function \(f\) if and only if \(f (a) = b\text{.}\)

  2. Each point on the graph of the function \(f\) has coordinates \((x, f (x))\) for some value of \(x\text{.}\)

  3. The vertical line test tells us whether a graph represents a function.

  4. We can use a graph to solve equations and inequalities in one variable.

Subsubsection 4.5.3 STUDY QUESTIONS

  1. How can you find the value of \(f (3)\) from a graph of \(f\text{?}\)

  2. If \(f (8) = 2\text{,}\) what point lies on the graph of \(f\text{?}\)

  3. Explain how to construct the graph of a function from its equation.

  4. Explain how to use the vertical line test.

  5. How can you solve the equation \(x + \sqrt{x} = 56\) using the graph of \(y = x + \sqrt{x}\text{?}\)
