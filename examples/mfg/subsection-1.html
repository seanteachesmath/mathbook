Subsection 1.1 Key Concepts

  1. We can describe a relationship between variables with a table of values, a graph, or an equation.

  2. Linear models have equations of the following form:

    \begin{equation*} y = (\text{starting value}) + (\text{rate of change}) \cdot x \end{equation*}

  3. The general form for a linear equation is \(Ax + By = C\text{.}\)

  4. We can use the intercepts to graph a line. The intercepts are also useful for interpreting a model.

  5. A function is a rule that assigns to each value of the input variable a unique value of the output variable.

  6. Function notation: \(y = f (x)\text{,}\) where \(x\) is the input and \(y\) is the output.

  7. The point \((a, b)\) lies on the graph of the function \(f\) if and only if \(f (a) = b\)

  8. Each point on the graph of the function \(f\) has coordinates \((x, f (x))\) for some value of \(x\text{.}\)

  9. The vertical line test tells us whether a graph represents a function.

  10. Lines have constant slope.

  11. The slope of a line gives us the rate of change of one variable with respect to another

  12. Formulas for Linear Functions.
    \begin{equation*} \begin{aligned}[t] \text{Slope:}\amp\amp m \amp =\dfrac{\Delta y}{\Delta x} = \dfrac{y_2 -y_1}{x_2 -x_1}\\ \amp\amp\amp = \dfrac{f(x_2) - f(x_1)}{x_2 -x_1}\\ \text{Slope-intercept form:}\amp\amp y \amp=b + mx\\ \text{Point-slope form:}\amp\amp y \amp=y_1 + m(x - x_1) \end{aligned} \end{equation*}

  13. The slope-intercept form is useful when we know the initial value and the rate of change.

  14. The point-slope form is useful when we know the rate of change and one point on the line.

  15. Linear functions form a two-parameter family, \(f (x) = b + mx\text{.}\)

  16. We can approximate a linear pattern by a regression line.

  17. We can use interpolation or extrapolation to make estimates and predictions.

  18. If we extrapolate too far beyond the known data, we may get unreasonable results.
